Useful commands

Here is a list of commands you might find useful.

Plugins

Lands (Claim system)

Accepting invites

/Lands accept <land>

Chat with land

/Lands chat [land] <message>

Claim chunks

/Lands claim

Create a land

/Lands create

Delete your land

/Lands delete

Deny invite

/Lands deny

Deposit money to land bank

/Lands deposit [land] <amount>

Enter edit mode for a land. Actions like /Lands claim will be executed for this land.

/Lands edit <land>

Display command usages of Lands

/Lands help [page]

Show information about a land

/Lands info [land]

/Lands invites

/Lands leave <Land>

/Lands map
